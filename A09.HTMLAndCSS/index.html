B-trees

What is a B-tree?

A B-tree is a balanced search tree that keeps keys sorted and supports search, insertion, and deletion in logarithmic time.

Requirements

  • For BTree(m):
  • Each Node can Have m children and m-l keys
  • Values and children are ordered
  • All Non-leaf nodes must have at most m children
  • All Leaves must be on the same level
  • Typically in cs400, m = 4, this is called a 2-3-4 Tree

    • Key features

  • Balanced height
  • High branching factor (many children per node)
  • Efficient for disk and SSD storage
  • B-Trees Grow Upwards
